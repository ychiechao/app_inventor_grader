# 超哥 App Inventor 學習幫手：目前功能紀錄

版本日期：2026-06-12  
備份分支：`codex/stable-before-batch-grading-2026-06-12`  
基準提交：`aedcc62`

## 系統架構

- 前端與 API 部署於 Cloudflare Pages。
- 正式網站：<https://app-inventor-grader.pages.dev/>
- Google Apps Script 負責存取 Google Drive 與 Google 試算表。
- OpenAI API 負責建立評分規準與分析 App Inventor `.aia` 原始檔。
- GitHub `main` 為正式開發版本；本分支保存加入批次評分前的穩定版本。

## 公開首頁

- 網站名稱為「超哥 App Inventor 學習幫手」。
- 顯示目前可供學生查看的作業列表。
- 作業可呈現草稿、排程、開放與關閉狀態。
- 學生可直接點選作業進入專屬繳交頁，不需自行貼上作業代碼。

## 教師管理

- 使用 Cloudflare 環境變數 `TEACHER_ACCESS_KEY` 保護教師管理頁。
- 建立作業時可輸入作業名稱、專案描述及選填的範例 `.aia`。
- AI 自動建立 100 分評分規準：
  - 功能介面需求：20 分
  - 程式邏輯完成度：50 分
  - 目標功能正確性：30 分
- 美工、命名與錯字原則上不扣分，除非影響功能或無法驗證。
- 可編輯作業名稱、描述、發布狀態、開放時間與截止時間。
- 可複製每份作業的學生繳交連結。
- 可查看學生繳交名單、三項分數、總分、時間與作業檔案。
- 可刪除作業；刪除前會二次確認，並將評分試算表、範例檔及學生作業檔移至 Google Drive 垃圾桶。

## 學生繳交

- 學生由作業專屬網址進入，不需輸入作業代碼。
- 支援「AI 初評」與「正式繳交」兩種模式。
- AI 初評不需填寫 Email、班級與座號，也不保存檔案或個人資料。
- 正式繳交需填寫 Email、班級與座號。
- 正式繳交檔名格式包含繳交日期、班級與座號。
- 相同 Email 再次正式繳交時：
  - 評分前先檢查既有紀錄。
  - 顯示是否覆蓋的確認訊息。
  - 取消時保留舊檔與舊評分，且不呼叫 AI 評分。
  - 同意後重新評分並更新原紀錄。
  - 新檔成功儲存後，舊檔才移至 Google Drive 垃圾桶。

## 評分結果

- 依序分段顯示功能介面需求、程式邏輯完成度、目標功能正確性。
- 每段顯示分數、完成證據、缺少或錯誤功能及具體改進建議。
- 顯示總分與整體評語。
- 評語文字採靠左對齊，便於閱讀較長內容。
- 正式繳交後將分數、評語、學生資料、檔案網址與 AIA 摘要寫入 Google 試算表。

## Google Drive 與試算表

- Apps Script 使用作業索引試算表管理公開作業 ID。
- 每份作業建立獨立的評分試算表。
- 舊版評分試算表可自動遷移至作業索引。
- 學生檔案與範例檔由 Apps Script 存入指定的 Google Drive 資料夾。
- Cloudflare 與 Apps Script 使用 `APPS_SCRIPT_TOKEN` 驗證請求。

## 驗證狀態

- 自動測試共 8 項，涵蓋建立作業、公開列表、作業狀態、初評、正式繳交、Email 重複確認、確認覆蓋及刪除作業 API。
- 正式首頁、教師頁、學生繳交頁與 Apps Script API 已完成線上連線驗證。

## 尚未實作
